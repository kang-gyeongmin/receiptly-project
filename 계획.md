## 📌 Phase A/B — AI 입력 재연결 (2026-07-02 착수)

챗봇으로 **자연어/영수증 캡처를 넣으면 파싱·자동분류해서 저장**. 무료 로컬 유지 + 확인 후 저장.

**자동분류 방식 (무료 로컬 "RAG식")**: ① 과거 내역에서 같은/유사 가게명의 최빈 카테고리(개인화 학습) → ② 키워드 힌트 → ③ 기본값. "확인 후 저장"에서 사용자가 고치면 그게 다시 ①의 학습 데이터가 됨.

## 0. 배경 (인터뷰 요약)

- **최우선**: 지금 깨져 있거나 가짜 데이터를 보여주는 화면을 실제로 동작하게 만들기
- **목표**: 포트폴리오/데모가 아니라 **실제로 내가/남이 쓸 수 있는 앱**
- **AI 방향**: 지금 방식(무료 로컬 EasyOCR + 정규식) 유지. 이번엔 신규 연결 안 함
- **죽은 코드(OCR·자연어 함수)**: 이번엔 삭제/보류. 나중에 연결
- **보안·안정성**: 이번 범위 아님. 단, 반드시 나중에 할 일이므로 아래에 기록만 남김
